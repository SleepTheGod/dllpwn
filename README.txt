To install this tool do all steps as followed

1. git clone https://github.com/SleepTheGod/dllpwn/

2. chmod +x dllpwn

3. cd dllpwn

4. chmod +x dllpwn.sh

5. bash dllpwn.sh

6. sudo mv dllpwn.sh /usr/local/bin/dllpwn

How to use this once you install

Just type dllpwn --help

Usage: dllpwn <process_to_inject> <init_data> [output_file]
       dllpwn --help | -h

Arguments:
  process_to_inject   Path to the process executable to inject the shared object into.
  init_data           Data to initialize the shared object with.
  output_file         (Optional) File to write the output to. Default is /tmp/test.txt.
root@chi:~#
